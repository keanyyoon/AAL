# Glim · Prototype (POC)

11~15세 글로벌 문해력 AI 튜터 **Glim**의 학습 플로우 프로토타입입니다.
세션 선택(학습목록) → 화성 지문 학습씬(소크라틱 문답) → 학습 리포트까지의 Happy Path.

## 어떻게 보나요

이 폴더는 **단독 실행형**입니다. HTML과 필요한 이미지가 한곳에 들어 있습니다.

1. 이 `prototype` 폴더를 통째로 내려받습니다.
   - 저장소 우측 상단 **Code ▸ Download ZIP** → 압축 해제 후 `prototype` 폴더로 이동
2. `index.html`을 더블클릭해 브라우저(크롬 권장)로 엽니다.
3. 학습목록에서 **"The Hard Landing on Mars"** 카드를 클릭 → 학습 시작.
   좌측 하단 **✦ 별**을 탭하면 학습자 발화 → Agent 소크라틱 응답이 순차 진행됩니다.

> GitHub는 HTML을 소스로만 표시하고 화면으로 렌더링하지 않습니다. 반드시 내려받아 `index.html`을 열어 주세요.
> 태블릿·PC 화면 기준으로 최적화되어 있습니다.
