# UX 브리프 (UX Brief)

- 작성일: 2026-07-01 · 작성: AAL UX 디자인팀 · 기반: 기획 PRD·시나리오, 브랜딩 세계관(Lumen Atlas)·비주얼

## 1. 디자인 원칙 (Design Principles)
1. **막히지 않게, 답은 주지 않게** — 소크라틱을 UX로 구현. 정체 감지 → 비계 노출 → 스스로 도달. 좌절 최소화.
2. **성장이 보이게** — 매 세션 성장 체감(SE), 매주 성장 서사(PT). "밝아지는 지도"로 시각화.
3. **몰입의 대비** — 세션은 저자극·집중, 리워드/성장은 빛·컬러 개방(감정 리듬).
4. **자율성** — 페르소나 선택, 진행 속도 자기 조절.
5. **태블릿·PC 우선, 반응형** — 정보 밀도는 태블릿/PC 기준, 모바일은 세로 스택·핵심만.
6. **접근성 기본값** — WCAG 2.1 AA, 터치 타깃 ≥44px, 색 외 형태·라벨 병행.

## 2. 디자인 테마 검토 (세계관 반영)
- 세계관 "Lumen Atlas(빛으로 밝히는 지도)"를 UI 은유로: 성장맵=밝아지는 대륙, 리워드=Lumen(빛), 3축=길/구조물/별자리.
- 세션 화면은 세계관 장식 최소화(읽기 집중), 온보딩·성장·리포트에서 세계관 시각 개방.

## 3. 반응형 전략
| 기기 | 세션 메인 레이아웃 | 비고 |
|---|---|---|
| PC/Mac | 지문+문항(좌) · AI 대화(우) 2단 | Primary |
| 태블릿 | 2단(가로) / 탭 전환(세로) | Primary |
| 모바일 | 세로 스택(지문→문항→대화), 대화는 하단 시트 | 반응형 |

## 4. 접근성 (WCAG AA 요약)
- 대비 4.5:1↑, 키보드 내비게이션, 포커스 표시, 스크린리더 라벨, 자막(듣기 활동), 애니메이션 감소 옵션, 아동 가독 폰트/행간.

## 5. UX가 4대 데이터에 기여하는 지점
- 완료율①: 진행 표시·저마찰 완료 버튼·좌절 방지 비계. 페르소나②: 선택·대화 UX. 리포트③: 열람 유도·섹션 위계. 학습패턴④: 자연스러운 상호작용으로 로그 확보.

## 6. 확장 훅
- 디자인 토큰(브랜딩과 공유)·컴포넌트 라이브러리로 다국어·다크모드·과목 확장 대비. 문자열 i18n 분리 전제.
