# UX 디자인 시스템 (Design System) — Sapiens · Dawn

- 작성일: 2026-07-01 · 작성: AAL UX·브랜딩팀 · 테마 Dawn of Civilization · 로고 여명(Dawn)
- 목적: 브랜딩 테마 토큰과 로고를 UX 컴포넌트로 확정 → 중충실도 프로토타입/개발 핸드오프의 단일 기준.

## 1. 디자인 토큰 (CSS Variables)
```
/* Color */
--ink-900:#0F1A33; --ink-700:#1B2A4A; --surface:#16213D; --surface-2:#1D2A49;
--line:#2C3A5E; --text:#EDE3CC(다크)/#1B2A4A(라이트); --muted:#9AA8CA;
--gold:#FFC24B;  --ember:#FF8A5B;  --parchment:#EDE3CC;  --mint:#37D6B3;
--axis-meaning:#FF8A7A; --axis-logic:#5B8CFF; --axis-knowledge:#B78CFF;
--success:#37D6B3; --warning:#FFB454; --danger:#FF6B81;
/* Radius */ --r-sm:8px; --r-md:12px; --r-lg:16px; --r-xl:22px(아이콘)
/* Space (4pt) */ 4/8/12/16/24/32/48
/* Elevation */ soft shadow; glow는 성취에만
/* Motion */ 180~260ms ease; prefers-reduced-motion 대응
```
- **모드:** 다크(Ink 배경/Parchment 텍스트) · 라이트(Parchment 배경/Ink 텍스트) — 토큰 스왑.
- **대비:** 본문 4.5:1↑. 3축은 색 + 형태(길/기둥/별) + 라벨 병행.

## 2. 타이포그래피
- 헤딩/디스플레이: 세리프(Fraunces / Noto Serif KR) — 문명·지성.
- 본문·UI: Inter + Pretendard(국문), 다국어(KR·TW 번체·VN).
- 지문(읽기): 가독 최우선(행간 1.8~1.9, 자간 여유). 스케일 태블릿/PC 기준, 모바일 축소.

## 3. 핵심 컴포넌트 스펙
| 컴포넌트 | 규칙 |
|---|---|
| 버튼(Primary) | 골드 배경/잉크 텍스트, r-md, 최소 44px 높이 |
| 버튼(Ghost) | 투명+라인, 보조 액션 |
| 지문 패널 | 저자극 서피스, 하이라이트=골드 20% |
| 대화 버블 | AI=서피스-2(좌), 학생=민트틴트(우), 발문/비계 톤 |
| 비계 힌트 | 단계 뱃지(①②③), "정답 없음" 원칙 |
| 리워드 칩 | Lumen(골드)·스트릭(🔥), glow 절제 |
| 성장맵 | 시대 진급(불→문자→도시→기계→우주) + 3축 게이지 |
| 리포트 카드 | 학생=눈높이/성취 모먼트, 학부모=담백·차트 |
| 시대 배지 | 세계관 진급 표시(현재 시대) |
| 알림 | 성과 시에만(카카오/웹푸시) |

## 4. 로고 사용 (요약)
- 세로형 기본 / 가로형(헤더) / 심볼(앱아이콘·파비콘). 초소형은 광선 생략.
- 컬러=골드 그라데이션, 저대비 배경엔 모노. 상세: 브랜딩 `로고_가이드_여명.html`.

## 5. 반응형
- PC/Mac·태블릿 Primary(세션 2단), 모바일 세로 스택(대화 하단 시트). 터치 타깃 ≥44px.

## 6. 접근성 (WCAG 2.1 AA)
- 대비·키보드·포커스·스크린리더 라벨·자막(듣기)·모션 감소·아동 가독.

## 7. 확장 훅
- 전부 토큰화 → 다크/라이트·다국어·과목 확장은 토큰/리소스 교체로. 시대 아이콘 체계는 국어·수학=새 시대/분과로 증설.
