# 화면 플로우 & 정보구조 (Flow & IA) — [온보딩·로그인 포함 v0.2]

- 작성일: 2026-07-01 · 작성: AAL UX 디자인팀

## 1. 전체 플로우
```
[진입] → 로그인/계정연동 → (최초)온보딩 → 홈(내 Atlas)
                                              │
                    ┌─────────────────────────┼───────────────────┐
                (학생) 세션 시작            성장맵 보기         페르소나 변경
                    │
     아이스브레이킹 → 세션 메인(SM) → 세션 종료+리포트(SE) → 리워드 획득 → 홈
                    │(막힘)
              소크라틱 발문 → 비계 사다리 → 스스로 도달

[학부모 앱] 로그인 → 데일리 로그(PD) ⇄ 4주 성장추이(PT) ⇄ 1:1 문의 / 성과 알림(카카오·앱)
```

## 2. 화면 목록 & IA
### 학생
1. **로그인/계정연동** — 청담 계정/간편 로그인. 로그인 시각 로깅(①). 최소 정보.
2. **온보딩(최초 1회)** — 세계관 인트로 → 탐험가 등록(닉네임/아바타) → 페르소나(길잡이) 선택 → 관심사 태그. (스킵/나중에 가능)
3. **홈(내 Atlas)** — 오늘의 세션 진입, 내 성장맵 요약, 스트릭, 획득 Lumen.
4. **아이스브레이킹** — 학습 외 관심사 대화(관계), 지난 레슨 복기 + 오늘 프리뷰.
5. **세션 메인(SM)** — 지문+문항 · AI 대화(발문·비계) · 진행 표시.
6. **세션 종료+리포트(SE)** — 오늘 배운 것 / 좋아진 점 / 보강점 + 리워드 + 다음 예고.
7. **성장맵/보상** — 3축(길·구조물·별자리) 성장, 배지, Lumen, 스트릭.
### 학부모
8. **학부모 로그인/홈**
9. **데일리 세션 로그(PD)** — 난이도·강약 유형·선호·완료여부.
10. **4주 성장 추이(PT)** — 추이 차트 + AI 코멘트 + 외부준거 매핑.
11. **성과 알림 수신 / 1:1 문의**

## 3. 정보 위계 원칙
- 세션은 "지문 읽기"가 시각 1순위, AI 대화는 조력자 위치. 리포트는 "좋아진 점(성장)"이 정서 정점 → 시각 강조.
- 학부모는 "안심(데일리)"과 "확신(4주 추이)"을 분리 제공.

## 4. 상태(빈/로딩/에러/막힘)
- 빈 상태: "첫 탐험을 시작해볼까?" / 막힘: 비계 단계 노출 + 격려(답 없음) / 에러: 데이터 유실 없이 재개.

## 5. 확장 훅
- 화면 문자열 i18n 분리, 외부준거 매핑은 테이블 참조. 홈=Atlas 구조는 과목(대륙) 확장 수용.
