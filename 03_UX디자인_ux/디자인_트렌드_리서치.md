# 디자인 트렌드 리서치 (Design Trends) — AI 튜터·에듀테크 2025~2026

- 작성일: 2026-07-01 · 작성: AAL UX·브랜딩팀 · 목적: 최신 디자인 트렌드를 Sapiens/Dawn 방향과 대조·정렬
- 성격: 2025~2026 공개 업계 자료 종합. 수치는 원문 인용(별도 검증 권장).

## 1. 7대 트렌드 요약
| # | 트렌드 | 요지 | Sapiens(Dawn) 적용/시사점 |
|---|---|---|---|
| T1 | **대화형 AI 인터페이스 표준화** | 실시간 Q&A·피드백, 읽기 쉬운 말풍선·타임스탬프·프로필, 음성 지원 | 세션의 소크라틱 대화 UX가 이미 정합. 단 "답 제공"이 아닌 발문형으로 차별 |
| T2 | **적응형·개인화의 가시화** | 진단→맞춤 경로, 실시간 난이도 조정. 개인화가 참여 최대 30%↑(원문) | 3축 프로파일·성장맵·적응형 비계로 구현. "보이는 문해력"과 결합 |
| T3 | **비주얼: 플랫 쇠퇴 → 깊이·다크 적응** | 제로뎁스 플랫 퇴조, 글래스모피즘·공간감(spatial), AI 맥락은 다크 기본 | Dawn = **다크 기본 + 여명 빛** 방향과 정합. 깊이는 절제(가독 우선) |
| T4 | **게이미피케이션 성숙** | 스트릭·XP·마스코트(Duo), 스트릭 프리즈로 이탈 완화. 7일 스트릭=장기잔존 3.6배(원문) | Lumen·스트릭·시대 진급·길잡이 캐릭터로 채택. ⚠️ Duo식 '죄책감 알림' 반작용은 회피 |
| T5 | **Calm / Transparent AI** | "감각적 화려함보다 인지적 명료성", 시각적 과장의 종말, "이게 노력을 줄이나?"를 질문 | 핵심 원칙 "읽기=저자극, 성취=빛 개방"과 정확히 일치. CHALK식 과장과 차별 |
| T6 | **크로스디바이스 연속성** | 폰→태블릿→PC로 끊김 없는 이어보기 | 태블릿·PC 우선 반응형 + 성장 로그 학생단위 영속(확장훅) |
| T7 | **신뢰·투명성(교사·학부모)** | AI 워크플로 신뢰, 학부모/교사 대상 투명 리포트 | 두 갈래 리포트(성취+사고), 성과 시에만 알림, 외부 준거 매핑 |

## 2. 세부 관찰
- **대화 UI**: 말풍선·타임스탬프·아바타가 사실상 표준 컴포넌트. 우리는 여기에 "비계 사다리·정답 미제공 가드레일"을 얹어 차별.
- **다크 적응**: AI 제품에서 밝은 화이트 라이트모드 기본이 다크 적응형으로 이동. Dawn의 딥 네이비 기본이 트렌드에 부합하면서 "밤→여명" 서사와 결합.
- **게이미피케이션의 양면**: Duolingo(2025 MAU 1.28억, 스트릭 프리즈로 이탈 21%↓ 원문)가 벤치마크지만, 마스코트의 '죄책감 유발 알림'은 아동·학부모 신뢰 리스크. → 우리는 **자기 성장(시대 진급)·성과 알림 절제**로 순화.
- **Calm design**: 2026 담론의 핵심이 "visual theatrics의 종말". CHALK의 3D·시네마틱 과시와 정반대 축에서 "따뜻한 지적 명료함"이 오히려 신선한 차별점.

## 3. 경쟁 비주얼 포지셔닝 (재확인)
- CHALK = 하이테크·시네마틱(감각적 화려함) / Duolingo = 캐주얼·마스코트(과잉 게임화 논란) / Khanmigo = 절제하나 밋밋.
- **Sapiens/Dawn = "따뜻한 지적 문명 × Calm·명료 × 절제된 게이미피케이션"** — 트렌드의 성숙 방향(T5)과 정합하는 빈자리.

## 4. 실행 체크리스트 (UX/개발 반영)
- [ ] 대화 UI 표준 컴포넌트(말풍선·아바타·타임스탬프) + 비계/가드레일 레이어
- [ ] 다크 기본 + 라이트(양피지) 토글, 성취 순간에만 빛/모션 개방
- [ ] 게이미피케이션은 자기 성장 중심(시대 진급·3축), 알림은 성과 시에만(죄책감 유발 금지)
- [ ] 크로스디바이스 이어보기(세션 재개), 성장 로그 영속
- [ ] Calm 원칙: 화면마다 "이게 인지 노력을 줄이나?" 자문
- [ ] 접근성(WCAG AA)·인지 명료성 우선
