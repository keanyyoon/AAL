## 2. CHALK 4.0 심층 분석 (최우선 벤치마크)

출처: FirstHabit 공식 제품 페이지(https://www.firsthabit.com/ko/product), CES 2026 보도자료 다수, Digit.in 해설 기사 등. 아래 내용은 원사 공식 페이지 및 보도자료 기반이며, 독립 검증되지 않은 성과 수치는 별도 표기함.

### 2.1 Visual LLM 3D 챗봇 구조 (Conversational Lecture)

- CHALK 4.0의 핵심은 **Visual LLM**(자체 개발 시각 특화 대형언어모델)로, "교육 세계 최초의 Visual LLM"이라고 자체 소개하고 있음.
- 실시간 1:1 대화형 수업을 **3D 비주얼 챗봇**으로 구현. 3D 학습자료와 애니메이션이 학생의 질문·반응에 맞춰 멀티모달로 상호작용.
- 텍스트, 수식, 이미지, 음성을 통합 처리하는 멀티모달 분석을 통해 범용 LLM 대비 "설명력·추론력"이 뛰어나다고 주장(원사 발표, 미검증).
- "Full Screen Mode"와 "Problem Solving Mode" 두 가지 수업 모드를 지원하며, "Cinematic Learning Experience"(영화적 학습 경험)를 표방 — 3D 시각자료·애니메이션으로 추상적 개념을 시각화해 몰입도를 높인다는 설계 철학.
- 기술적 기반은 **Ontology 기반 프레임워크**로, "최고의 강사 사고 과정을 재현"한다고 설명(구체적 알고리즘·데이터 구조는 비공개).

### 2.2 AI 선생님 페르소나 10종

공식 페이지에서 확인된 10종 페르소나와 성격/설명 스타일:

| 페르소나 | 성격 | 설명 스타일 |
|---|---|---|
| Marie | 다정함 | 깔끔한 설명 |
| LEO | 유쾌함 | 심층탐구형 |
| SOPHIA | 차분함 | 쉬운 비유형 |
| MAX | 활기참 | 빠른 진도 |
| Ella | 세심함 | 질문세례형 |
| KAI | 참신함 | 실생활 탐구형 |
| Nora | 꼼꼼함 | 단계별 학습형 |
| Owen | 파워풀 | 스토리텔링형 |
| Lina | 사려깊음 | 수준 맞춤형 |
| Ethan | 명쾌함 | 도전적 과제형 |

**재참여 유도 방식(분석):**
- 학생이 페르소나 + 페이셜 아바타를 직접 선택하게 함으로써 "내가 고른 나만의 선생님"이라는 **소유감(ownership)**을 형성 — 이는 캐릭터에 대한 애착을 통해 재방문 동기를 만드는 전형적 게임화 기법.
- 페르소나별로 설명 스타일이 명확히 다르므로(예: Ella의 "질문세례형" vs Owen의 "스토리텔링형"), 학생의 학습 성향·기분에 맞춰 **같은 개념도 다른 방식으로 재학습**할 수 있어 반복 학습의 지루함을 줄이는 설계로 보임(추정).
- CHALK AI 엔진이 선택된 페르소나에 맞춰 수업 콘텐츠 자체를 커스터마이징한다고 명시 — 단순 스킨(외형) 변경이 아니라 **콘텐츠 전달 방식의 실질적 변화**를 준다는 점이 특징.
- 페르소나 시스템 자체가 소크라틱 발문 여부를 보장하지는 않음 — 발문 스타일(질문세례형 Ella 등)은 일부 존재하나, 전반적으로 "설명력 극대화"에 초점이 맞춰져 있어 AAL처럼 "답을 주지 않는 것" 자체가 설계 원칙인지는 불명확(추정).

### 2.3 Brain Map (온톨로지 그래프 기반 지식 시각화)

- 공식 명칭은 "Brain Scan"과 "Brain Map"으로 구성: **Brain Scan**은 학습자의 지식을 실시간 진단하는 "두뇌 스캐너" 역할, **Brain Map**은 그 결과를 사고 과정 그래프로 시각화.
- "사고 과정을 온톨로지 그래프로 시각화하여 내가 아는 것, 모르는 것, 앞으로 나아가야 할 방향을 명확히 제시"한다고 설명.
- 즉 개념(노드) 간 선후관계·의존관계를 온톨로지(지식 그래프) 형태로 구조화하고, 학생의 문제 풀이 데이터를 이 그래프에 매핑해 "마스터한 노드/미마스터 노드"를 색상·형태로 구분하는 방식으로 추정됨(세부 UI·알고리즘 비공개).
- **문해력 튜터에 응용 가능성(분석):**
  - 문해력 영역에서도 "의미 이해 → 논리적 추론 → 배경지식 연결"이라는 개념 간 의존관계가 존재하므로, AAL의 의미(Meaning)·논리(Logic)·지식(Knowledge) 3축 방법론을 **온톨로지 그래프 형태의 "리터러시 맵"으로 시각화**하는 아이디어는 구조적으로 이식 가능성이 있음(추정).
  - 다만 수학/STEM은 정답이 명확한 노드형 지식 구조(예: "분수 개념을 알아야 비율을 이해")로 온톨로지화가 비교적 쉬운 반면, 문해력의 "추론 능력", "행간 읽기" 같은 역량은 노드/엣지로 명확히 구조화하기 어려운 비정형적 성격이 강해 **그대로 이식하기보다 재설계가 필요**해 보임(추정, AAL 차별화 지점이 될 수 있음).

### 2.4 Fast Track Finder (최단 학습경로)

- "학생의 데이터와 학습 패턴을 분석해, 가장 효율적인 학습 루트를 자동으로 설계"하는 기능으로, "유치원부터 대학까지" 전 구간을 포괄한다고 설명.
- Brain Map(현재 지식 상태)과 결합해 "다음에 무엇을 배워야 목표에 가장 빨리 도달하는지"를 계산하는 경로 탐색(pathfinding) 알고리즘으로 추정.
- 목표 지향적 학습(예: 특정 시험·학년 목표)에 강점이 있는 구조로, AAL처럼 "사고력 자체의 성장"을 목표로 하는 경우에는 "최단 경로"라는 개념보다 "충분한 사고 과정을 거치는 것"이 더 중요할 수 있어 철학적으로 상충 가능성이 있음(분석 포인트).

### 2.5 Gamified LMS "Brain Map Universe"

- 컨셉: "내 머릿속을 도시로, 학습을 게임으로." 학생이 개념을 학습할수록 **Brain Map Universe**(두뇌 우주도시)가 확장됨.
- 메커니즘 상세(공식 페이지 기준):
  1. **우주도시 건설**: 학습이 깊어질수록 관념이 별·행성처럼 자리 잡고, 도시가 우주 속에서 성장. 즉 학습한 개념 하나하나가 시각적 자산(건물/행성)으로 누적됨.
  2. **캐릭터·아바타 탐험**: 학생이 아바타를 선택하고, 미션을 클리어하며 Brain Map을 탐험 — 학습 과정 자체가 "퀘스트"로 재구성됨.
  3. **포인트·리워드**: 학습 과정에서 포인트를 얻고 이를 리워드로 교환. 포인트는 "실제 화폐처럼 사용 가능"하다고 언급됨(원사 발표) — 즉 게임 내 경제 시스템처럼 설계.
  4. **관광객 보너스 루프**: "화려하게 성장한 도시는 더 많은 관광객을 불러들여 보너스 포인트를 얻는다"는 세컨더리 루프 존재 — 성실한 학습(도시 성장)이 추가 보상(관광객)을 낳는 **선순환 구조**를 만들어 지속적 참여를 유도.
  5. **랭킹/소셜**: 랭킹 시스템으로 친구와 성취 공유·경쟁, 협력 요소도 있어 "개인 경험을 넘어 사회적 즐거움으로 확장"한다고 설명.
- 종합 평가: Duolingo의 스트릭/리더보드보다 한 단계 더 나간 "메타버스형 성장 시뮬레이션" 게이미피케이션으로, 단순 포인트 적립이 아니라 **누적된 학습 이력이 영구적으로 눈에 보이는 세계(우주도시)로 남는다**는 점이 특징. 다만 이 구조가 실제 학습 동기부여로 이어지는지, 혹은 도시 꾸미기 자체가 목적이 되어 학습에서 주의를 분산시키는지는 검증되지 않음(추정, AAL 설계 시 유의점).

### 2.6 멀티디바이스 및 글로벌 커리큘럼

- **멀티디바이스**: PC, 태블릿, 스마트폰뿐 아니라 "로봇"까지 지원한다고 명시. SLLM(소형 특화 LLM)과 온디바이스 모델을 활용해 네트워크 연결 여부와 무관하게 동일 경험을 제공한다고 주장.
- **글로벌 커리큘럼**: 공식 페이지에 소개된 예시는 다음과 같음.
  - 미국 정규 교과(Common Core 기반 수학·과학·영어·사회)
  - 미국 AP(Calculus, Physics, Chemistry 등)
  - 수학: 미적분, 한국 수능형 수학(대수·기하·확률과 통계)
  - 에세이 작성: TCR(Topic-Comment-Reasoning) 스타일 — **논리적 글쓰기 훈련 요소**로 문해력과 접점이 있는 부분
  - 국어: 수능 국어 문학 분석(비유·상징·주제 중심)
  - 과학: 물리(역학), 화학(유기화학)
  - 컴퓨터과학: Python 코딩 기초
  - 글로벌 스탠다드: IB, SAT 준비
  - 사회: 경제학(글로벌+한국)
- 커리큘럼의 압도적 다수가 **수학/STEM/시험 대비** 영역에 집중되어 있고, 언어·문해력 관련 콘텐츠는 "문학 분석"과 "TCR 에세이" 두 가지 정도로 상대적으로 비중이 작음.

### 2.7 완강률 76.4% 주장의 맥락

공식 페이지에 명시된 원문(번역):
> "CHALK의 효과는 실제 테스트를 통해 검증되었습니다. **2024년 한국과 보스턴에서 진행된 베타테스트**에서는 자기주도 학습 완강률이 **76.4%**에 달했으며, 이는 업계 평균의 5배 이상이었습니다. **2025년 블라인드 테스트**에서는 CHALK의 대화형 강의를 사용한 학생들이 최상위 학원의 강의 대비 **30~40% 더 높은 성과**를 보였습니다."

확인된 조건 및 한계:
- 시기: 2024년 8~9월경, 한국(장소 미상)과 미국 보스턴 두 지역에서 진행된 **베타테스트** 기준.
- 비교 대상: "기존 한국 인터넷강의 평균 완강률 15%"와 비교(일부 뉴스 기사에서 언급). 즉 오프라인 학원이 아니라 **인터넷 강의(VOD) 완강률**과 비교한 수치.
- 표본 크기, 참여자 선정 방식, 완강 기준(어느 수준까지 학습해야 "완강"으로 집계하는지), 독립 3자 검증 여부는 공개 자료에서 확인되지 않음.
- "+35%"(강의 블라인드 테스트, 최상위 온·오프라인 강의 평가점수 평균 대비) 수치도 별도 테스트(2025년, 블라인드)에서 나온 것으로 76.4%(2024년 베타테스트)와는 **다른 실험**의 결과임 — 두 수치를 혼용하지 않도록 주의 필요.
- 결론: 76.4%라는 수치는 **자사 베타테스트 기준의 자체 발표 수치**로, 독립기관 검증 근거는 확인되지 않음. "인터넷강의 완강률"이라는 낮은 기준선(15%) 대비 비교라는 점도 고려해야 함. (원사 발표, 미검증 — 벤치마크·투자자 자료 인용 시 이 맥락을 함께 표기 권장)

### 2.8 CHALK는 문해력(소크라틱 사고력 유도)을 다루는가

**결론: 부분적으로만 다루며, AAL과 같은 "정답을 주지 않는 소크라틱 발문" 자체를 핵심 설계 원칙으로 표방하지는 않는 것으로 보임(추정 포함 분석).**

근거:
1. CHALK AI의 공식 설계 철학은 "단순 정답이 아닌 사고와 설명이 함께하는 몰입형 학습"이며, "학습자의 이해 수준에 따라 설명 방식과 시각 자료를 실시간으로 조정"하는 데 방점이 있음. 이는 **"더 잘 설명해주는 AI"**에 가깝고, Khanmigo나 AAL이 표방하는 **"스스로 답을 찾도록 질문으로 유도하는 AI"**와는 방향이 다름.
2. CES 2026 현장 반응으로 언급된 "정답을 바로 알려주기보다 사고의 흐름을 따라가게 해주는 점이 인상적" 이라는 평가는 존재하나, 이는 추론 과정의 **시각화**(3D 애니메이션으로 풀이 단계를 보여줌)에 대한 인상일 가능성이 높고, 실제로 AI가 "질문만 던지고 답은 학생이 찾게 하는" 구조인지는 원문에서 명확히 확인되지 않음.
3. 커리큘럼 구성상 문해력 특화 트랙(비판적 읽기, 논증, 추론 훈련을 전담하는 코스)은 확인되지 않고, "국어" 관련 콘텐츠는 수능 문학 분석과 TCR 에세이 작성 정도로 제한적. 이는 수학/STEM에서 시작해 문해력으로 확장 중인 것으로 추정되며, "문해력이 주력인 AAL"과는 출발점이 반대임.
4. **AAL과의 핵심 차별점**:
   - AAL: 문해력(Literacy)이 코어 — 의미·논리·지식 3축의 소크라틱 발문으로 "생각하게 만드는" 것이 최종 목표. 정답 제공을 의도적으로 지연/회피.
   - CHALK: 수학/STEM/시험대비가 코어 — Visual LLM으로 "더 잘 이해시키는" 것이 목표. 설명력·추론 시각화가 핵심 가치제안이며, 최종적으로는 정답에 빠르고 정확하게 도달하도록 돕는 구조에 가까움(추정).
   - 즉 CHALK는 **"어떻게 하면 더 잘, 더 빠르게 이해시킬까"**에 최적화되어 있고, AAL은 **"어떻게 하면 학생이 스스로 사고하게 만들까"**에 최적화되어 있어, 표면적 기능(AI 챗봇, 개인화, 게이미피케이션)은 유사해 보여도 **근본 철학이 다름**. 이 차이가 AAL의 포지셔닝 메시지에서 명확히 강조되어야 할 지점.

## 3. 확장 훅(Expansion Hooks)

이번 벤치마크가 AAL의 글로벌(대만·베트남) 및 국어·수학 확장 논의에 시사하는 점:

1. **글로벌 확장 관점**: CHALK는 미국 시장(2026년 정식 출시 예정)과 SAT/AP/IB 등 국제 커리큘럼을 통해 "글로벌 학습 OS"를 표방하며 CES라는 글로벌 무대를 활용해 브랜드력을 빠르게 키우고 있음. 반면 대만·베트남 등 아시아 volume 시장으로의 직접 진출 근거는 검색되지 않음(추정: 아직 미국/한국 중심). AAL이 대만 테스트베드 → 베트남 볼륨 전략을 취한다면, CHALK가 아직 선점하지 않은 **아시아 로컬 언어·교육과정 문해력 영역**에서 선점 기회가 있을 수 있음.
2. **국어(문해력) 확장 관점**: CHALK의 국어 콘텐츠가 "수능 문학 분석" 수준에 머물러 있다는 점은, AAL이 표방하는 "정답을 주지 않는 소크라틱 발문 기반 문해력"이 명확한 백지 시장(백지 포지셔닝)일 가능성을 시사함. 단, Khanmigo의 Writing Workshop/Debate가 이미 유사 철학으로 4~10학년(9~16세)을 타깃하고 있어, 글로벌 시장에서는 Khanmigo가 AAL의 가장 직접적인 철학적 경쟁자임을 벤치마크 근거로 삼을 수 있음.
3. **수학 확장 관점**: CHALK, Synthesis 모두 수학/STEM에서 강력한 게이미피케이션·개인화 노하우를 이미 시장에서 검증받고 있음(Synthesis는 SpaceX 발 스토리로 브랜드력, CHALK는 CES 수상으로 신뢰도 확보). AAL이 향후 수학으로 확장한다면 이 두 제품이 직접 경쟁자가 되므로, "문해력에서 시작해 사고력 기반 수학(서술형·논증형 수학)으로 확장"하는 시나리오가 차별화된 진입 경로가 될 수 있음(정답 중심 수학이 아닌 "왜 그런가"를 묻는 수학).
4. **게이미피케이션 설계 참고**: Brain Map Universe(우주도시 건설)와 Duolingo의 스트릭/리더보드는 검증된 참여 유도 메커니즘이나, AAL의 소크라틱 철학과 화려한 게임화가 상충하지 않도록 "사고 과정 자체를 보상하는" 게이미피케이션(예: 좋은 질문을 만들었을 때, 스스로 답을 찾아냈을 때 보상)을 설계할 필요가 있음 — 단순 정답/진도 기반 보상은 소크라틱 철학과 충돌 위험이 있음.
5. **리포트(학부모용) 참고**: 지니아튜터의 "단원별·성취기준별·교과역량별" 시각화 대시보드, CHALK의 "Personalized Dashboard"는 학부모 신뢰 확보에 리포트가 핵심임을 보여줌. AAL도 "사고력이 어떻게 성장했는가"를 정성+정량으로 보여주는 학부모 리포트 설계가 Q4 파일럿(청담 재원생 100명)의 핵심 성공 요인이 될 수 있음.

## 4. 한계 및 후속 조사 제안

- CHALK 4.0의 정확한 가격 정책, 실제 사용자 리뷰(초기 정식 출시 전이라 제한적), 소크라틱 발문 여부에 대한 3자 검증 자료는 확보하지 못함. 정식 출시(2026년 하반기로 추정) 이후 재조사 필요.
- 완강률 76.4% 등 원사 발표 수치는 독립 검증되지 않았으므로, 사내 자료에 인용 시 반드시 "원사 발표, 자사 베타테스트 기준"이라는 조건을 함께 표기할 것을 권장.
- 대만·베트남 시장에 특화된 AI 튜터(로컬 스타트업 포함)는 이번 검색 범위에서 확인하지 못했음 — 별도의 현지어 검색(중국어 번체, 베트남어) 리서치가 필요.
