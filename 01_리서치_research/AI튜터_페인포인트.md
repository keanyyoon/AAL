# 현존 AI 튜터 페인포인트 (User Pain Points with Existing AI Tutors)

- 작성일: 2026-07-01
- 작성: AAL 리서치팀
- 목적: 타깃 사용자(11~15세 학생 + 학부모)가 현존 AI 튜터(CHALK, Khanmigo, 콴다, Duolingo, Speak 등)에서 **실제로 느끼는 불만**을 구조화하고, 각 페인포인트에 **AAL이 공략할 지점**을 매핑 → 기획팀 차별화의 직접 입력.
- 성격: 벤치마크 조사·사용자 리뷰 패턴·업계 통설을 종합. 정량 수치가 필요한 항목은 "추가 검증 필요"로 표기.

---

## A. 학생이 느끼는 페인포인트

| # | 페인포인트 | 실제 양상 / 근거 | 어느 제품에서 두드러지나 | **AAL 공략 지점** |
|---|---|---|---|---|
| S1 | **"답만 알려줘서 실력이 안 는다"** | 정답·풀이를 즉시 주는 앱은 베끼기·의존을 유발. 스스로 사고하는 힘이 안 자람. AI 시대에 답 접근 비용이 0이 되며 가치 붕괴(콴다 사례) | 콴다, 일반 챗봇(ChatGPT류) | 소크라틱 발문으로 **답을 주지 않고 사고를 끌어냄** — 존재 이유 자체가 이 페인포인트 해결 |
| S2 | **"금방 질린다 / 재미없다"** | 초기 신기함 후 이탈. 반복 드릴·무미건조한 챗봇 대화는 청소년 주의지속에 취약. 재참여(리텐션) 실패가 카테고리 공통 난제 | Khanmigo(재참여 약점 추정), 대다수 텍스트형 튜터 | 게이미피케이션 + 세계관(브랜딩) + 페르소나 애착 + **매 세션 성장 체감**으로 재미·재방문 설계 |
| S3 | **"내가 얼마나 늘었는지 모르겠다"** | 성장이 눈에 안 보이면 유능감·동기 저하. 점수만 나오고 "무엇이 어떻게 나아졌는지" 서사가 없음 | 회화앱·문제풀이앱 다수 | **성장 가시화**(성장맵·배지·세션 전후 비교) + "지난번 근거 1개→오늘 3개" 식 구체 피드백 |
| S4 | **"어려우면 막히고, 쉬우면 지루하다"(난이도 미스핏)** | 적응형이 약하면 좌절 또는 권태. 비계가 없으면 막힌 채 방치됨 | 정적 콘텐츠형, 비계 약한 소크라틱 | 적응형 출제 + **점감 비계 사다리**(막히면 힌트 단계적 제공, 답은 끝까지 안 줌) |
| S5 | **"AI가 나를 모른다 / 기계적이다"** | 획일적 톤, 맥락 기억 부재로 관계감 부족. 청소년은 자율성·관계성에 민감 | 단일 톤 챗봇 | 페르소나 선택(소유감) + Endorse 톤 + 아이스브레이킹(학습 외 관심사)으로 관계 형성 |
| S6 | **"영어가 어려워서 생각 자체를 못 한다"(EFL 부하)** | 영어로 짜내려다 사고가 축소됨. 이해 붕괴 어휘에서 좌절 | 영어 전용 앱 | **"점화·심층 사고는 한국어, 습득·시험은 영어"** 언어 원칙 — EFL 특유 교란 정면 대응 |

## B. 학부모가 느끼는 페인포인트

| # | 페인포인트 | 실제 양상 / 근거 | 어느 제품에서 두드러지나 | **AAL 공략 지점** |
|---|---|---|---|---|
| P1 | **"아이가 뭘 하는지, 늘고 있는지 안 보인다"** | 진행상황 불투명 → 결제 지속 근거 없음. 오프라인 학원도 강사 인상평 수준에 그침 | 다수(학부모 리포트 빈약) | **두 갈래 리포트**(성취 축 + 사고 축), 데일리 로그 + 4주 성장 추이 + AI 코멘트 |
| P2 | **"점수는 오르는데 진짜 실력인지 모르겠다"** | 요령·암기로 오른 점수 불신. 본질 역량 성장 확인 욕구 | 문제풀이·점수보장형 | 성취(시험 역량)와 **문해력(본질) 성장**을 함께 제시, 전환 관계 명시 |
| P3 | **"알림이 너무 많거나 / 없거나"** | 과도한 알림은 피로, 없으면 방치감. 균형 실패 | 다수 앱 | **성과 포인트에서만 발송**(새 난이도 첫 이해, 3축 성장, 주간 스트릭) — 피로 최소화 |
| P4 | **"믿을 만한가?(효과·근거)"** | 근거 없는 능력 향상 주장에 대한 불신(루모시티 FTC 제재·뉴턴 몰락) | 검증 안 된 에듀테크 | 준거 외부화(CEFR·TOEFL Jr 정렬) + **학술 검증(KATE)** + 지식그래프 특허 자산 |
| P5 | **"결국 사교육을 대체 못 한다"** | 앱은 보조일 뿐, 성적·입시로 이어지는 확신 부족 | 대다수 학습앱 | 수능·내신·수행평가 **전환의 명시**(사고 행동 → 시험 역량 매핑), 청담 재원생 연계 |
| P6 | **"가격 대비 가치가 불투명"** | 구독 지속 이유 부재 시 이탈 | 구독형 다수 | 매주 리포트가 "성장의 서사"를 전달 → **구독 갱신의 이유**로 전환 |

## C. 페인포인트 → AAL 차별화 요약 (기획팀 입력)

1. **핵심 3대 페인포인트(S1·S3/P1·P4)가 곧 AAL의 3대 무기다:**
   - "답만 준다" → **소크라틱(답을 안 준다)**
   - "성장이 안 보인다" → **보이는 문해력(성장 가시화 + 두 갈래 리포트)**
   - "못 믿겠다" → **학술·외부 준거 검증(KATE·CEFR·TOEFL Jr)**
2. **재참여(S2)는 카테고리 공통 실패 지점** — 여기서 이기면 카테고리에서 이긴다. 게이미피케이션·세계관·페르소나 애착이 파일럿 데이터 ②(페르소나 반응)의 핵심.
3. **경쟁 구도:** CHALK는 "설명력·비주얼·게이미피케이션"이 강하나 "답을 주지 않는 소크라틱·문해력 본질"은 불명확(추정). Khanmigo는 소크라틱 철학은 같으나 재미·성장 가시화·로컬라이즈가 약함(추정). → **AAL의 빈자리 = "소크라틱 문해력 × 재미·성장 가시화 × 글로벌 로컬라이즈"의 교집합.**

## 확장 훅 (Expansion Hooks)
- 위 페인포인트는 국어·수학 확장 시에도 대부분 유효(특히 P1·P2·P4·P6). 국어는 P2("진짜 실력") 소구가 더 강해질 수 있음(모국어 문해력 = 학부모 불안 본진).
- 글로벌: P4(신뢰)·P5(입시 연결)의 준거가 지역별로 다름(대만=독해 소양 공교육 가치, 베트남=국제학교·영어 투자). 페인포인트의 "무게"가 지역마다 달라짐 → 시장별 메시지 우선순위 조정 필요.
