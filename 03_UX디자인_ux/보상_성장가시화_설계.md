# 보상 · 성장 가시화 설계 (Rewards & Growth Visualization) — [v0.2 추가]

- 작성일: 2026-07-01 · 작성: AAL UX 디자인팀 · 원칙: 보상은 "문해력 사고 행동"에 묶는다(외적보상 함정 회피)

## 1. 보상 철학
- **Pointsification 금지:** 포인트만 뿌리지 않는다. 보상은 근거 대기·텍스트 복귀·사실/의견 구분 등 **관찰된 사고 행동**과 성장에 연동.
- **자기 대비 기본값:** 또래 경쟁(랭킹)은 파일럿 기본 OFF(11~15세 위축 방지). 자기 성장 중심.

## 2. 리워드 루프 (세계관: Lumen)
| 행동 | 보상 | 세계관 표현 |
|---|---|---|
| 세션 완료 | Lumen(빛) 획득 | "오늘의 땅을 밝혔다" |
| 사고 행동 발생(근거 대기 등) | 보너스 Lumen | "실마리를 스스로 찾았다" |
| 새 난이도 첫 이해 | 배지 + 성과 알림 | "새 지형 개척" |
| 주간 연속 학습 | 스트릭 유지 | "탐험 연속 일수" |
| 3축 중 성장 | 성장맵 확장 | 길/구조물/별자리 밝아짐 |

## 3. 성장 가시화 (핵심 = "밝아지는 Atlas")
- **내 Atlas(성장맵):** 학습할수록 지도가 밝아지고 넓어짐. 3축을 시각 요소로 분리:
  - 의미=**길(Path)** 이 이어짐 / 논리=**구조물(Bridge)** 이 세워짐 / 지식=**별자리(Constellation)** 가 연결됨.
- **세션 전후 비교(SE):** "지난번 근거 1개 → 오늘 3개" 같은 구체 델타를 시각·문구로.
- **3축 성장 게이지:** 의미/논리/지식 각각의 성장(비대칭 정상 — 3축 프로파일).
- **마일스톤:** 새 난이도·새 스킬 획득 시 축하 모먼트(빛 번짐 애니메이션, 절제).

## 4. 완료율 KPI와의 연결 (①)
- 진행 링/남은 스텝 표시로 "거의 다 왔다" 유도 → 완료율↑.
- 종료 시 성장 확인 + 리워드 → 재접속 동기(②) → 스트릭 → 완료율 지속.

## 5. 학부모용 성장(투명성)
- 학부모 리포트에도 성장맵 요약(항해일지) 반영 → "진짜 실력이 자란다"(P2) 소구.

## 6. 접근성·안전
- 애니메이션 감소 옵션. 보상이 과금·경쟁 압박이 되지 않도록. 실패에 벌점 없음(성장 지향).

## 7. 확장 훅
- Lumen·배지·스트릭 로직을 과목 비종속 데이터로. 국어·수학=새 대륙으로 성장맵 확장.
