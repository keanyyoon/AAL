# Lessons Learned (실수 반영 로그)
> 프로젝트 지침: 실수를 하면 동일 실수를 반복하지 않도록 여기에 반영한다.

## 2026-07-01
- HTML 리포트는 세로 스크롤 대신 **PPT식 페이지네이션**(슬라이드+방향키+점 네비)으로 만든다. (유저 선호)
- 가치 구조 핵심 메시지: "성장하는 재미를 위해 들어와 머문다." (이전 "점수를 위해 들어와…"에서 변경)
- 세계관은 단일안이 아니라 **복수 시안 비교**로 의사결정 지원(문명진화/세계탐험/마을연결 3안).
- GitHub 아카이빙: 연결 가능한 GitHub MCP가 레지스트리에 없어 **로컬 git 초기화 + push 명령 안내** 방식으로 대체. 워크스페이스가 마운트라 git object 정리 경고가 나오나 커밋에는 영향 없음.
