## 3. 디자인 원칙 (5)
1. **밤 → 여명**: 딥 네이비 배경에서 골드 빛이 번지는 성장 은유.
2. **따뜻한 지성**: 클래식 문명 모티프를 모던 미니멀로 (차가운 하이테크 지양 = CHALK 차별).
3. **몰입의 대비**: 읽기(세션)는 저자극·집중, 성취·성장은 빛·컬러 개방.
4. **연령 존중**: 11~15세를 어리게 취급하지 않는 세련미 + 안전.
5. **Calm·명료 우선**: "이 요소가 인지 노력을 줄이나?"를 항상 질문. (2026 트렌드 정합)

## 4. 디자인 토큰 (그대로 복사해 사용)
```css
/* Color */
--ink-900:#0F1A33;   /* 배경(밤) */
--ink-700:#1B2A4A;   /* 서피스/텍스트(라이트) */
--surface:#16213D; --surface-2:#1D2A49; --line:#2C3A5E;
--parchment:#EDE3CC; /* 뉴트럴(양피지) · 다크모드 텍스트 */
--lumen-gold:#FFC24B; /* 빛·성취·리워드·브랜드 */
--ember:#FF8A5B;      /* 불꽃·강조 */
--mint:#37D6B3;       /* 완료·긍정 */
/* 3축 */
--axis-meaning:#FF8A7A; --axis-logic:#5B8CFF; --axis-knowledge:#B78CFF;
/* 상태 */
--success:#37D6B3; --warning:#FFB454; --danger:#FF6B81;
/* Radius */ --r-sm:8px; --r-md:12px; --r-lg:16px; --r-icon:22px;
/* Space(4pt) */ 4 / 8 / 12 / 16 / 24 / 32 / 48
/* Motion */ 180~260ms ease; 성취 순간에만 glow; prefers-reduced-motion 대응
```
- **모드:** 다크(Ink 배경/Parchment 텍스트) 기본, 라이트(Parchment 배경/Ink 텍스트) 토글.
- **대비:** 본문 4.5:1↑. 3축은 색 + 형태(길/기둥/별) + 라벨 병행.

**타이포:** 헤딩/디스플레이 = 세리프(Fraunces / Noto Serif KR) · 본문·UI = Inter + Pretendard(다국어 KR·TW 번체·VN) · 지문(읽기)은 가독 최우선(행간 1.8~1.9).

## 10. 디자인 시 반드시 지킬 제약 (Guardrails)
- **소크라틱 UX**: AI는 정답을 직접 주지 않는다 → 화면·카피가 "발문·비계"로 유도.
- **성장 가시화**: 매 세션 성장 체감 + 매주 성장 서사. "밝아지는 Atlas"로.
- **보상 ≠ 남발**: Lumen·배지·스트릭은 사고행동에 연동. 또래 경쟁 기본 OFF.
- **Calm**: CHALK식 3D·시네마틱 과시로 흐르지 않기. 따뜻한 지적 명료함.
- **접근성**: WCAG AA, 3축 색+형태 병행, 모션 감소 옵션, 아동 안전·연령 적합.
- **확장 훅**: 문자열 i18n 분리, 토큰화 유지(다국어·과목 확장 대비).
