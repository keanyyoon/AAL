# 사용자 시나리오 (User Scenarios / Journeys) — [v0.2 추가]

- 작성일: 2026-07-01 · 작성: AAL 기획팀 · 용도: UX팀 화면 플로우·브랜딩 세계관의 직접 입력

## 페르소나 (파일럿 기준)
- **학생 — 지호(13세, 중1):** 청담 재원생. 영어는 중상위지만 "왜 그렇게 읽는지"는 약함. 게임·캐릭터 좋아함, 쉽게 지루해함. 자율성 원함.
- **학부모 — 지호 母(43세):** 점수 불안 + "진짜 실력" 궁금. 학원 진행상황이 안 보여 답답. 알림 과다에 피로. 스마트폰으로 확인.

---

## 시나리오 1 — 학생 첫 경험 (온보딩 → 첫 세션)
1. **온보딩:** 지호가 링크 접속 → 세계관 인트로("너의 이야기가 시작되는 곳") → 페르소나 선택(내가 고른 나만의 튜터, 소유감) → 관심사 태그(축구·게임) 입력.
2. **로그인:** 청담 계정 연동/간편 로그인(로그인 시각 기록 = 데이터①).
3. **아이스브레이킹:** 튜터가 관심사(축구)로 말 걸기 → 긴장 완화, 관계 형성.
4. **첫 세션(SM):** 지문 읽기 → 문제. 막힘 → 튜터가 "정답" 대신 **"글의 어느 부분이 그렇게 말하지?"** 발문 → 지호가 텍스트로 복귀(사고행동 로깅) → 스스로 실마리 발견.
5. **세션 종료(SE):** 완료 버튼 → "오늘 배운 것 / 지난번보다 근거를 2개 더 찾았어(↑) / 다음엔 반박에 답해보자(→)" → 성장 체감 + 리워드 획득.
**정서 곡선:** 낯섦 → 흥미(캐릭터·관심사) → 살짝 막힘 → 스스로 해냄(유능감) → 성장 확인(뿌듯).

## 시나리오 2 — 학생 재참여 (2주차, 리텐션 핵심)
1. 성과 알림 or 스트릭 리마인더 → 재접속.
2. 접속 복기: "지난번 네가 찾은 근거 기억나?" → 연속감.
3. 성장 가시화(성장맵/배지 확장) 확인 → "더 하고 싶다" 동기.
4. 세션 진행 → 스트릭 유지 → 종료 시 새 성취.
**측정:** 페르소나·Endorse 톤별 재접속률 차이(데이터②).

## 시나리오 3 — 학부모 일상 확인 (데일리 → 4주)
1. **데일리(PD):** 저녁, 앱 열람(무알림) → 오늘 난이도·강약 유형·완료 여부 확인 → "혼자 풀어냈네" 안심.
2. **성과 알림:** "지호가 처음으로 B1 난이도 지문을 이해했어요" 카카오 알림 → 열람 → 뿌듯 → 재방문.
3. **4주 추이(PT):** 성장 그래프 + AI 코멘트("8주차부터 추론형 문항 개선 두드러짐") + CEFR/등급 매핑 → **"점수만이 아니라 진짜 읽는 힘이 자란다"** 확신 → (파일럿 후) 구독 갱신 의향.
**측정:** 열람률·알림 오픈율·섹션 체류(데이터③).

## 시나리오 4 — 막힘·좌절 관리 (엣지 케이스)
- 지호가 발문에도 계속 막힘 → 비계 사다리 자동 상향(①조건 표시→②결과 표시→③연결 힌트) → 그래도 안 되면 격려 톤으로 난이도 조정. **끝까지 정답은 주지 않되 방치하지 않음.** → 이탈 방지.

## 시나리오 5 — 확장 미리보기 (밑그림)
- (국어) 지호가 영어에서 기른 "근거 대기"가 국어 비문학에서도 되는지 확인 → 두 과목 기록 연결.
- (글로벌) 대만/베트남 학생: 동일 플로우, 언어·준거만 로컬라이즈(TOEFL Jr·현지 시험).

## UX/브랜딩 인계 포인트
- 온보딩·로그인·아이스브레이킹·성장 가시화·리워드·학부모 리포트가 **하나의 세계관**으로 꿰어져야 함(브랜딩 세계관 컨셉과 연결).
- 정서 곡선의 정점은 "스스로 해냄 + 성장 확인" — 모든 화면이 이 순간을 향해 설계.
