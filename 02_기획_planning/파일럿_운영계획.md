# 파일럿 운영 계획 (Pilot Operations Plan) — Q4 2026

- 작성일: 2026-07-01 · 작성: AAL 기획팀

## 1. 개요
- 대상: 청담 재원생 **약 100명** · 기간 **8~12주** · 판매 없음(무료 리서치) · 주 2회 세션 권장.
- 목적: 4대 데이터 확보로 PMF 신호와 설계 가설 검증.

## 2. 페르소나 A/B 설계 (데이터 ②의 핵심)
- **방식:** 4주에 한 번 페르소나 로테이션(같은 학생이 서로 다른 톤 경험) + 코호트 간 비교. 파일럿 단계 페르소나 2개(최소) ~ 8개(다양 톤 탐색) 운영 후 선호도 높은 톤 고도화.
- **측정:** 페르소나별 세션 완료율 · 학습 성취도 · 세션 체류시간 · 종료 후 설문(선호도).
- **가설:** 접속 시 복기 + 종료 시 Endorse 톤이 다음 세션 재참여를 높인다(칸미고 실패 가설 검증).

## 3. 측정 설계 (Measurement Plan)
| 데이터 | 지표 | 수집 화면 | 성공 기준(Phase 0) |
|---|---|---|---|
| ① 완료율 | 세션 완료율(주 2회), 완료까지 평균시간 | SM·SE | 60%↑ |
| ② 페르소나 | 페르소나별 완료율·성취·체류·설문 선호 | SM·SE | 유의미한 선호 차이 규명 |
| ③ 리포트 반응 | 열람률, 알림 오픈율, 섹션별 체류시간, 성과알림 후 재방문율 | PD·PT | 반응 높은 섹션 규명 |
| ④ 학습 패턴 | 지문 난이도 추이, 강·약 유형 변화, 선호 이동 | SM→PT | 4주 단위 변화 관찰 가능 |

## 4. 운영 흐름 (주차별)
- **W0(준비):** 계정 발급·로그인, 온보딩, 페르소나 배정, 기준선(baseline) 세션.
- **W1–4:** 페르소나 세트 A, 데일리 로그 누적, 4주차 1차 추이.
- **W5–8:** 페르소나 로테이션(세트 B), 2차 추이 비교.
- **W9–12(선택):** 3차 추이, 원거리 전이 점검, 종료 설문·인터뷰.

## 5. 피드백 채널
- 학부모 페이지 내 **1:1 문의 채널** 개설 → 실시간 피드백 수렴.
- 학생: 세션 종료 설문(경량), 이탈자 대상 별도 인터뷰.

## 6. 운영 리스크 & 완화
- **저조한 접속 지속:** 성과 알림·스트릭·성장 가시화로 재방문 유도. 이탈 조기경보(2회 연속 미접속).
- **데이터 누락:** 완료 버튼=로그 확정 구조로 무결성 확보(개발팀 검증).
- **소크라틱 답 유출:** 검증팀 상시 QA로 가드레일 점검.
- **아동 개인정보:** 최소 수집·동의·보호자 관리(검증팀 필수 점검).

## 7. 파일럿 후 의사결정
- 선호 페르소나 톤 확정 → 고도화. 반응 높은 리포트 섹션 → Phase 1 강화. 학습 패턴 신호 → 적응형·레벨 체계 근거. PMF 신호 시 튜터 선택 UX·보상 시스템·초기 진단 착수.
