# AAL — AI Agent Learning (Creverse AI Tutor)

11~15세 글로벌 학생을 위한 **문해력(Literacy) 중심 AI 튜터**. "답을 주는 대신 소크라틱(Socratic) 발문으로 스스로 사고하게" 만들고, 문해력의 성장을 매일 보이게 한다. 의미·논리·지식 3축(디오픈/THE OPEN, KATE 인증) 기반. Web(태블릿·PC primary, 반응형).

## 당면 목표
2026 Q4 청담 재원생 100명 파일럿 MVP(8~12주). 확보 데이터: 세션 완료율 / 페르소나 반응 / 학부모 리포트 반응 / 학습 패턴.

- Round 1: 6개 팀 산출물 완료(총 27종). 유저 결정 대기(세계관 3안, 페르소나 수, 네이밍, 콘텐츠 공급, 기술 방향).
