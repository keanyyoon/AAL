# 세계관 v2 — "어둠 속의 빛 (Light in the Dark)" · 서비스명 **Glim**

- 작성 2026-07-08 · 브랜딩팀 · **기존 Sapiens / 문명의 진화(Ages) 세계관 폐기, 신규 컨셉으로 대체**
- **결정(2026-07-08): 서비스명 = Glim** (glimmer, 어둠 속 작은 빛). 배경은 더 딥한 near-black(#03050B~#060911).
- 원칙(유저 지시): ①Dark BG + 아주 작은 별빛 몇 개 ②어둠 속 빛(Light)이 주 비주얼 키 ③Agent·주요 버튼도 빛 컨셉 ④비주얼은 과하지 않게 — **주인공은 문제 콘텐츠** ⑤서비스명 제안

---

## 1. 한 줄 컨셉
> **"읽고 생각한다 = 어둠 속에서 빛을 켠다."**
> 모르는 것은 어둠, 이해·사고의 순간은 빛. 학생이 스스로 생각을 밝힐 때마다 어둠 속에 작은 빛(별)이 하나씩 켜진다.

- 소크라틱 철학과 정합: Agent는 답(빛)을 대신 켜주지 않는다. **학생이 스스로 빛을 켜도록** 곁에서 은은히 비추는 동행하는 빛.
- 이전 '문명 진화'의 화려한 게임 세계관과 달리, **미니멀·정적·집중**. 화면은 대부분 어둡고 조용하며, 콘텐츠(지문·문제)만 또렷이 밝다.

## 2. 왜 이 컨셉인가
- **주인공은 콘텐츠:** 어두운 배경 = 시각 소음 제거 → 지문·문제가 유일하게 밝은 대상 = 시선 집중(인지부하↓). "세계관이 콘텐츠를 가리지 않는다."
- **성장의 은유가 자연스럽다:** 학습할수록 어둠에 별이 늘어난다 = 성장 가시화(과하지 않게). 밤하늘이 촘촘해지는 조용한 성취.
- **Calm·다크 기본** = 2026 디자인 트렌드 및 "읽기=저자극" 원칙과 정합. CHALK의 요란한 3D와 정반대 축.
- **글로벌·연령 중립:** 빛·별은 문화 보편적, 유아틱하지 않음.

## 3. 비주얼 키 (Visual Key) — 레퍼런스 반영(프리미엄·시네마틱)
> 참고: "VOID"형 다크 프리미엄 목업(웜골드↔쿨블루 대각 광선 + 날카로운 4-포인트 별빛) + 잔잔한 밤하늘 스타필드.

| 요소 | 정의 |
|---|---|
| **배경** | near-black 딥 네이비 `#080B14` → `#0B1020`. 균일하게 어둡고 깊게. |
| **대각 광선(Aurora Beam)** | 화면 한쪽(예: 좌하단)에서 대각선으로 스며드는 빛 띠. **웜 골드/오렌지 `#FFB25A`↔쿨 블루 `#5B8CFF`/시안 `#8FD3FF` 그라데이션**. 크게·부드럽게 블러, 저휘도. 시네마틱 깊이감. |
| **별(star)** | 두 종류: ①아주 작은 점(1~2px, 저투명, 느린 twinkle) 다수 ②**날카로운 4-포인트 스파클**(십자 플레어 + 코어 글로우) 2~4개만 포인트로. |
| **빛(Light) = 주 키** | 부드러운 원형 글로우(코어 웜 화이트 `#FFF3D6` → 골드 헤일로). Agent·성취·주요 버튼에 사용. |
| **콘텐츠** | 가장 밝게(고대비). 지문 카드는 **글래스모피즘**(반투명 `rgba(20,26,43,.6)` + 얇은 광택 테두리 + blur). **화면에서 가장 빛나는 것 = 콘텐츠.** |
| **텍스트/타이포** | 헤딩: 라이트 산세리프(300) + **이탤릭 세리프 강조**("advanced *technology.*" 식). 라벨: 대문자 + 넓은 자간(letter-spacing .25~.35em). 본문/캡션: 모노 계열 or 넓은 자간의 테크니컬 톤. 본문 `#E7EAF3`/보조 `#8A93AC`. |
| **정답·성취** | 새 별(4-포인트 스파클)이 '켜지는' 순간의 빛 번짐(절제, 1회). |
> 팔레트: 어둠(딥네이비) · 빛(웜화이트→골드) · **대각 광선(골드↔블루)** · 텍스트(오프화이트). 광선·스파클은 은은하게, 콘텐츠 가림 금지.

## 4. AI Agent = "동행하는 빛" (Companion Light)
- 캐릭터/얼굴 아바타가 아니라 **반짝이는 추상적 파형 구체(shimmering waveform sphere)**. 특정 모양 없이 은은히 맥동·미세 파동(breathing/soft ripple). 말할 때 파형이 살짝 살아남.
- 평소: 화면 구석에서 **아주 약하게** 빛나며 대기(막히기 전 개입 안 함).
- 말할 때: 부드럽게 밝아지며 곁으로. 발화 끝나면 다시 은은해짐.
- 학생이 스스로 답을 밝히면 Agent 빛이 잠깐 커지며 축하 → 곧 다시 물러남.
- 의인화 최소(불쾌한 골짜기 회피). "빛"이라 문화·언어 중립, 페르소나 5종은 **빛의 색온도/움직임 결**로 구분(예: 따뜻한 노랑↔차분한 은백).

## 5. 버튼 & 주요 컴포넌트 (빛 컨셉 적용)
- **Primary(주 행동):** 빛 자체 — 웜 글로우로 채운 버튼(soft glow). "빛을 누른다".
- **Secondary:** 어둠 위 얇은 빛 테두리(outline glow), 채움 없음.
- **비활성:** 빛이 꺼진 상태(테두리만 희미).
- **입력/선택지:** 어두운 캡슐, 선택 시 가장자리에 빛이 스민다(오답은 빛이 사그라들 뿐, "틀렸다" 문구 없음).
- **진행/성장:** 점(별)들이 하나씩 켜지는 인디케이터.
- **원칙:** 빛 효과는 **행동 가능·성취 지점에만**. 나머지는 어둠으로 물러나 콘텐츠를 돋보이게.

## 6. 적용 강도 (과하지 않게)
- 배경 별·글로우는 **은은하게**(저채도·저휘도), 애니메이션 느리게. 모션 감소 옵션 필수.
- 학습(세션) 화면: 세계관 장식 최소 → 지문·문제가 유일한 주인공. 성취·홈·리포트에서만 빛을 조금 더 개방.
- "밤하늘"은 분위기이지 놀이터가 아니다. 게임적 요소는 최소.
