# 디자인 테마 (Design Theme) — "Dawn of Civilization"

- 작성일: 2026-07-01 · 작성: AAL 브랜딩·UX팀 · Sapiens · 세계관 A
- **로고 확정(2026-07-01): 시안 C 여명(Dawn) 마크** — 지평선 위 떠오르는 빛. 상세 가이드: `로고_가이드_여명.html`.
- 한 줄: **밤(미지)에서 여명(밝혀짐)으로 진화하는 따뜻한 지적 문명.** 하이테크(CHALK)와 캐주얼(Duo) 사이의 빈 자리.

## 1. 테마 원칙
1. **밤 → 여명:** 딥 네이비 배경에서 골드 빛이 번지며 "밝혀지는" 성장 은유.
2. **따뜻한 지성:** 클래식 문명 모티프를 모던 미니멀로. 차가운 네온 대신 온기.
3. **몰입의 대비:** 읽기(세션)는 저자극·집중, 성취·성장은 빛·컬러 개방.
4. **연령 존중:** 11~15세를 어리게 취급하지 않는 세련미 + 안전.

## 2. 컬러 토큰 (Design Tokens)
| 토큰 | 값 | 용도 |
|---|---|---|
| `--ink-900` (딥 네이비) | `#0F1A33` | 기본 배경(밤) |
| `--ink-700` | `#1B2A4A` | 패널/서피스 |
| `--lumen-gold` | `#FFC24B` | 브랜드·빛·성취·리워드 |
| `--ember` | `#FF8A5B` | 강조·불꽃(시대 시작) |
| `--parchment` | `#EDE3CC` | 뉴트럴(양피지·라이트 서피스/텍스트) |
| `--mint` (성취) | `#37D6B3` | 긍정 피드백·완료 |
| 3축 | 의미 `#FF8A7A` · 논리 `#5B8CFF` · 지식 `#B78CFF` | 성장 가시화 |
| 상태 | 성공 민트 · 주의 앰버 `#FFB454` · 위험 소프트레드 `#FF6B81` | 아동 친화 톤다운 |

- 라이트 모드: parchment 배경 + ink 텍스트. 다크 모드: ink 배경 + parchment 텍스트. (토큰 스왑)
- 대비: 본문 4.5:1↑ 준수. 3축은 색+형태(길/기둥/별) 병행.

## 3. 타이포그래피
| 역할 | 서체 방향 | 비고 |
|---|---|---|
| 디스플레이/헤딩 | 클래식 세리프(예: Fraunces/Noto Serif) | 문명·지성감 |
| 본문·UI | 휴머니스트 산세리프(Inter) + 국문 Pretendard | 가독·다국어 |
| 지문(읽기) | 가독 최우선, 넉넉한 행간/자간 | 세리프 옵션 검토 |

## 4. 형태·아이콘 시스템
- 라운드 12~16px, 부드러운 그림자, glow는 성취/리워드에만 절제.
- **시대 아이콘 세트**: 🔥불 · 📜문자 · 🏛️기둥/도시 · ⚙️톱니 · 🚀로켓 — 일관된 라인 스타일. 성장맵·진급 배지에 사용.
- 3축 시각 심볼: 의미=길/문자, 논리=기둥/다리, 지식=별자리.

## 5. 모션 원칙
- 절제된 easing, "빛이 번지는" 트랜지션은 성취 순간에만. 모션 감소 옵션 필수.

## 6. 확장 훅
- 모든 값은 토큰으로 관리 → 다크/라이트·다국어·과목 확장 시 토큰만 교체.
- 시대 아이콘 체계는 국어·수학 확장 시 "새 시대/분과"로 증설.
