# 비주얼 방향 (Visual Direction) — 무드 · 컬러 · 타이포

- 작성일: 2026-07-01 · 작성: AAL 브랜딩팀 · UX 디자인 시스템과 정합(공유 토큰)

## 1. 무드보드 방향
- 키워드: **밝은 지적 어드벤처 / 빛과 지도 / 따뜻한 미래감 / 몰입되되 산만하지 않은**.
- 레퍼런스 톤(방향): 별·지도·빛 모티프, 부드러운 그라데이션, 종이지도+디지털의 결합. 과한 3D·다크판타지 지양.
- CHALK가 '우주·회로기판' 톤이라면 AAL은 **'빛이 번지는 지도/별자리'** 톤으로 차별.

## 2. 컬러 (제안 토큰)
| 역할 | 색 | 용도 |
|---|---|---|
| Primary(빛) | 딥 블루 `#1B2A4A` + 루멘 옐로우/골드 `#FFC24B` | 브랜드·성장·리워드(빛) |
| Accent(탐험) | 틸/민트 `#37D6B3` | 성취·긍정 피드백 |
| 3축 컬러 | 의미=코럴, 논리=블루, 지식=퍼플 | 성장 가시화 3축 구분 |
| 배경 | 딥 네이비 → 밝아지는 그라데이션 | "밝혀진다" 은유 |
| 상태 | 성공=민트, 주의=앰버, 위험=소프트 레드 | 톤 다운(아동 친화) |
> 접근성: 본문 대비 WCAG AA(4.5:1) 이상. 3축 컬러는 색맹 대비 형태·라벨 병행.

## 3. 타이포그래피
- 국문: Pretendard / 본고딕 계열(가독성·다국어). 영문: Inter 계열.
- 지문(읽기)용은 **가독성 최우선**(충분한 행간·자간, 세리프 옵션 검토). UI는 산세리프.
- 태블릿·PC 기본, 모바일 반응형 스케일.

## 4. 컴포넌트 톤
- 라운드(부드러움) 12~16px, 부드러운 그림자, 빛 번짐(glow) 포인트를 리워드·성장에만 절제 사용.
- 세션 화면은 몰입 위해 **저채도·저자극**, 리워드·성장 화면에서 **빛·컬러 개방**(감정의 대비).

## 5. 확장 훅
- 컬러·타이포를 **디자인 토큰**으로 정의(UX와 공유) → 다국어·다크모드·과목 확장 시 토큰만 교체.
- 3축 컬러 체계는 국어·수학에도 재사용.

> **결정 요청:** 컬러/무드 방향 승인 → UX 디자인 시스템에 토큰으로 반영.
