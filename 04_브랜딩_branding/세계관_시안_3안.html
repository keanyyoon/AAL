시안 A 문명의 진화 — Ages of Lumen

고대 원시문명에서 시작해 문자·도시·기술·우주 문명으로 발전한다. 읽어낼수록 나의 문명이 다음 시대로 진화.

🎮 성장 가시화

  • 시대(Age) 진급: 불 🔥 → 문자 📜 → 도시 🏛️ → 기계 ⚙️ → 우주 🚀
  • "내가 문명을 끌어올린다"는 강한 진보감

🧭 3축 매핑

  • 의미 = 언어·문자의 발명
  • 논리 = 건축·법·기술 체계
  • 지식 = 문명 간 지혜의 축적

✅ 장점

  • 진보 서사 강함 → 장기 동기·성취감
  • 역사·지식 주제와 자연 결합
  • 확장: 국어·수학 = 새 기술분과

⚠️ 리스크

  • 선형 진보 → 자유 탐험감 약함
  • 초기 원시 단계가 "낮은 수준"으로 느껴질 수
  • 정복·경쟁 톤 배제 필요

vs CHALK: 우주도시(공간 축적)와 달리 "시간·진화" 축으로 차별.

시안 B 세계 탐험 보드 — Grand Voyage

전 세계를 돌며 각 지역의 "텍스트 세계"를 밝힌다. 지역 → 지역으로 나아가며 대륙을 수집(부루마블형).

🎮 성장 가시화

  • 방문·개척한 지역 수, 세계지도가 밝아짐
  • 여권·스탬프·랜드마크 수집

🧭 3축 매핑

  • 의미 = 현지 언어·표현
  • 논리 = 여정 경로·문제 해결
  • 지식 = 지역 문화·배경지식

✅ 장점

  • 다양성·수집 재미
  • 글로벌(대만·베트남) 정체성과 직접 부합
  • 지역=주제 → 원거리 전이 측정과 궁합

⚠️ 리스크

  • 보드게임 주사위=운 요소가 "실력"과 충돌 → 실력 기반 이동으로 설계
  • 깊이보다 넓이로 흐를 위험

vs CHALK: 지식 축적형과 달리 "세계 탐험·다양성" → 글로벌 소구 강점.

시안 C 마을에서 도시로 — Story Isles

허허벌판 섬에서 시작해, 읽어내며 건물·마을을 짓고 섬과 섬을 다리로 연결해 도시로 키운다(심시티형).

🎮 성장 가시화

  • 내 섬/도시가 자라고 섬들이 연결됨
  • 건물=습득 스킬, 다리=지식 연결

🧭 3축 매핑

  • 의미 = 건물(어휘·구문)
  • 논리 = 도로·다리(연결·구조)
  • 지식 = 지구·도시 구획(스키마)

✅ 장점

  • 건설·연결 은유가 문해력의 "연결·재구성"과 정확히 일치
  • 자유도·소유감 큼
  • 상상력·서사력 확장 여지

⚠️ 리스크

  • CHALK "두뇌 우주도시"와 톤이 가장 근접 → 차별화 부담
  • 건설 관리가 학습 시간 잠식하지 않도록 절제

채택 시: "도시 소유"가 아니라 "지식의 연결(다리)"에 초점을 둬 차별.

권고 & 결정 요청

  • 글로벌 정체성 + 차별성 최우선 → B (세계 탐험)
  • 문해력 은유 정합 + 성장 체감 → A (문명 진화) 또는 C (마을 연결)
  • 검증팀: C는 CHALK와 톤이 가장 근접해 차별화 부담 → A·B가 차별성에서 유리

세 안 모두 상위 은유 "Lumen(빛)"과 결합 가능 — A=시대를 밝히는 불꽃, B=지도를 밝히는 등불, C=섬을 밝히는 등대.

결정 요청: A / B / C 중 채택(또는 하이브리드). 선택 후 페르소나·비주얼·성장맵을 해당 세계관으로 정합.
