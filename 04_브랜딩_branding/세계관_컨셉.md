# 세계관 컨셉 (Worldview / Universe Concept) — [v0.2 핵심 산출물]

- 작성일: 2026-07-01 · 작성: AAL 브랜딩팀 · 기반: 기획 전략·시나리오, 리서치(CHALK "두뇌 우주도시" 대비)

## 0. 왜 세계관인가
게이미피케이션·성장 가시화·페르소나·리워드가 제각각이면 학생은 "기능 묶음"을 경험한다. **하나의 세계관이 이 요소들을 하나의 이야기로 꿰면**, 학습은 "여정(journey)"이 되고 재참여(S2)와 성장 체감(S3)이 자연스러워진다. CHALK의 "두뇌 우주도시(Brain Map Universe)"가 지식을 별·행성으로 쌓는 공간 은유라면, AAL은 **"읽는 힘으로 세계를 밝혀가는" 탐험 서사**로 차별화한다.

## 1. 세계관 대전제
> **"모든 텍스트는 하나의 세계다. 너는 읽어냄으로써 그 세계를 밝히는 탐험가다."**

- 문해력 = 세계를 읽고 밝히는 힘. 지문 하나하나가 안개에 싸인 미지의 땅.
- 학생은 **답을 받는 학생이 아니라, 스스로 실마리를 찾는 탐험가(Explorer)**. → 소크라틱 철학과 세계관이 일치.
- 튜터(페르소나)는 정답을 주는 선생이 아니라 **길잡이 동료(Guide/Companion)**. → "답을 주지 않는다"가 세계관 설정상 당연해진다.

## 2. 추천 컨셉: "The Lumen Atlas" (가칭) — 빛으로 그리는 지도
- **은유:** 세상은 아직 그려지지 않은 거대한 지도(Atlas). 학생이 텍스트를 읽고 의미·논리·지식을 밝힐 때마다 지도의 한 조각에 **빛(Lumen)**이 들어온다.
- **성장 = 지도가 밝아짐.** 성장 가시화가 "내 지도(내 세계)가 넓어지고 밝아지는" 경험으로 직결(유능감).
- **3축의 세계관적 번역:**
  - 의미(Meaning) → **길(Path)**: 단어·청크가 이어져 길이 된다.
  - 논리(Logic) → **다리·구조물(Structure)**: 주장-근거·추론이 세계를 지탱한다.
  - 지식(Knowledge) → **별자리(Constellation)**: 읽은 것들이 연결되어 큰 그림이 된다.
- **왜 좋은가:** ①문해력의 본질(읽어 밝힌다)과 은유가 정확히 맞음 ②CHALK의 '도시 건설'과 톤이 겹치지 않음(도시 소유물 축적 vs 세계를 밝히는 탐험) ③국어·수학 확장 시에도 "새 대륙"으로 자연 확장 ④글로벌 문화 보편적(지도·빛·별은 문화 중립적).

### 대안 컨셉 (선택지)
- **B. "이야기 정원(Story Garden)":** 읽을수록 자라는 정원 — 서사력·상상력 강조에 유리하나 남학생 소구·확장성 다소 약함.
- **C. "생각의 항해(Voyage of Thought):** 미지의 바다를 항해 — 탐험 톤은 유사하나 '읽기=밝힘' 은유가 Atlas만큼 직접적이지 않음.
> **권고: A(Lumen Atlas)**. B/C 요소(성장·서사)는 A 안에 흡수 가능.

## 3. 세계관이 각 요소를 어떻게 꿰는가 (UX·개발 인계)
| 제품 요소 | 세계관 번역 |
|---|---|
| 온보딩 | "탐험가 등록 + 첫 지도 받기" |
| 페르소나 | "길잡이 동료를 고른다" (세계관 속 캐릭터) |
| 세션 = 지문 | "안개 낀 미지의 땅을 밝히기" |
| 소크라틱 발문 | 길잡이의 "이 길은 어디로 이어질까?" 질문 |
| 비계 사다리 | 길잡이가 건네는 단서(등불 밝기 조절) |
| 리워드/포인트 | "Lumen(빛)" 획득 |
| 성장 가시화 | 내 Atlas가 밝아지고 넓어짐 (3축=길·구조물·별자리) |
| 스트릭 | "탐험 연속 일수" |
| 학부모 리포트 | "탐험 기록(항해일지)" — 어디를 밝혔고 어떻게 성장했는지 |
| 확장(국어·수학) | "새로운 대륙"의 개척 |

## 4. 세계관 톤 (분위기)
- 경이(wonder)·발견(discovery)·따뜻함. 화려한 과시가 아니라 **"내가 밝혀냈다"는 조용한 성취감**.
- 다크·판타지 과잉 지양(11~15세 안전·부모 신뢰). 밝고 지적인 어드벤처.

## 5. 확장 훅
- 페르소나=세계관 속 길잡이로 추상화 → 과목·언어 확장에도 동일 세계관 유지.
- "새 대륙" 구조로 국어·수학 확장 자연 수용.
- Lumen(빛)/Atlas(지도) 명칭은 다국어 번역 부담 낮음(글로벌).

> **결정 요청:** 세계관 컨셉 A(Lumen Atlas) 채택 여부. 채택 시 네이밍·비주얼·페르소나가 이 위에서 구체화됨.
