# 디자인 리서치 (Design Research) — Sapiens

- 작성일: 2026-07-01 · 작성: AAL 브랜딩·UX팀 · 서비스명 가안 **Sapiens** · 세계관 A(문명의 진화)
- 목적: 비주얼 방향/테마 결정을 위한 레퍼런스·원칙 정리

## 1. 컨셉 앵커
- **Sapiens = 생각하는 인간**(homo sapiens). "답을 받는 게 아니라 스스로 사고해 문명을 진화시키는 존재".
- 세계관 A: 🔥불 → 📜문자 → 🏛️도시 → ⚙️기계 → 🚀우주. **시간·진보**의 축.
- 상위 은유 Lumen(빛): 시대를 밝히는 불꽃 → 성취·성장의 시각 언어.

## 2. 비주얼 레퍼런스 방향 (무드)
| 축 | 방향 | 이유 |
|---|---|---|
| 클래식 × 모던 | 고대 문명 모티프(기둥·성좌·양피지)를 **현대적 미니멀**로 재해석 | 유치하지 않게, 지적 신뢰(학부모 P4) |
| 밤 → 여명 | 딥 네이비(밤·미지) → 골드 빛(밝혀짐) 그라데이션 | "읽어서 밝힌다"·성장 가시화 |
| 재질감 | 돌·양피지 뉴트럴 + 빛 번짐(glow) 절제 | 문명의 물성 + 디지털 경이 |
| 아이콘 | 시대 심볼(불꽃·문자·기둥·톱니·로켓)을 일관된 라인 스타일 | 세계관 진급의 시각 체계 |

## 3. 경쟁 비주얼 언어 (차별점)
- **CHALK**: 우주·회로기판·3D 하이테크, 차가운 네온. → 우리는 **따뜻한 문명·빛**으로 차별(하이테크 과시 대신 지적 온기).
- **Khanmigo/Khan**: 절제된 교육 UI, 비주얼 임팩트 약. → 우리는 세계관으로 정서·몰입 부여.
- **Duolingo**: 밝은 플랫·마스코트 중심, 유아~전연령. → 우리는 11~15세가 "어리게 취급받지 않는" 지적 톤.
- **결론**: "따뜻한 지적 문명(warm intellectual civilization)"이 빈 자리. 하이테크(CHALK)와 캐주얼(Duo) 사이.

## 4. 타깃(11~15세 글로벌) 감성
- 유아틱함 거부, 자기 정체성·성취 중시 → 세련되되 과하게 무겁지 않게.
- 캐릭터 애착 유효(페르소나=시대의 현자) but 과잉 게임화는 학부모 신뢰↓ → **재미와 진지함의 균형**.
- 문화 중립 모티프(빛·별·문명 유물)는 KR·TW(번체)·VN 공통 소구.

## 5. 접근성·기술 제약 (WCAG 2.1 AA)
- 본문 대비 4.5:1↑, 3축 컬러는 색 외 형태·라벨 병행.
- 지문(읽기) 가독성 최우선: 충분한 행간·자간. UI는 산세리프, 헤딩은 디스플레이 세리프 옵션.
- 다크/라이트 겸용 토큰, 모션 감소 옵션, 태블릿·PC 우선 반응형.
- 다국어 폰트(Pretendard/본고딕 + 라틴 Inter/serif) — i18n.

## 6. 디자인 시사점 → Theme
- Theme는 "밤에서 여명으로 밝아지는 문명" = 딥 네이비 베이스 + 골드(빛/불꽃) + 돌/양피지 뉴트럴 + 3축 컬러.
- 헤딩=클래식 디스플레이 세리프(문명·지성), 본문·UI=휴머니스트 산세리프(가독).
- 상세는 `디자인_테마.md` 및 `브랜드_디자인_시안.html` 참조.

## 출처/검증
- 경쟁 비주얼은 각 사 공식 페이지 기반. 무드 레퍼런스는 방향 제시이며, 실제 시안·사용성은 UX 프로토타입/테스트로 검증 예정.
