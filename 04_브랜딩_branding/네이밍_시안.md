# 서비스 네이밍 시안 (Naming Concepts)

- 작성일: 2026-07-01 · 작성: AAL 브랜딩팀 · 기준: 세계관 A(문명의 진화) + 학습·문해력 의미
- 기존 'Lumo'는 폐기(유저 피드백). 문명(Civilization)·시대(Epoch)·배움(Learning)·지혜(Wisdom)·읽는 힘(Literacy)을 담는 방향.
- ※ 모든 후보는 **상표·도메인·기존 브랜드 충돌 검토 필요**(아래 리스크 표기는 사전 참고용).

## 1. 추천 Top 3
| 이름 | 읽기 | 어원·의미 | 세계관(문명진화) 연결 | 톤 |
|---|---|---|---|---|
| **Aevum** (에이범) | AY-vum | 라틴어 '시대·영겁(age)' | 문명이 시대를 거쳐 진화 = 세계관 핵심 은유 직결 | 지적·클래식·유니크 |
| **Sapio** (사피오) | SAH-pee-oh | 라틴어 '나는 안다·분별한다'(homo sapiens) | 생각하는 인간=문명+사고력, 소크라틱과 정합 | 스마트·성인 신뢰감 |
| **Epoca** (에포카) | eh-PO-ka | 이탈리아/스페인어 '시대(epoch)' | 시대의 진급=성장 가시화, 글로벌 발음 쉬움 | 세련·글로벌 |

## 2. 전체 후보 (컨셉별)

### 문명·시대(Age/Epoch/City)
| 이름 | 의미 | 메모 |
|---|---|---|
| **Aevum** | 라틴 '시대·영겁' | 세계관 직결, 유니크. 발음 안내 필요 |
| **Epoca** | 시대(epoch) | 글로벌 발음 쉬움, 부드러움 |
| **Aetas** | 라틴 '시대·세대' | 짧고 강함, 다소 생소 |
| **Polis** | 그리스 '도시국가'(문명·학문의 터) | 강하나 일반명사, 조합 추천(예: Lexipolis) |
| **Forge** | '벼려 만들다'(문명 건설) | 강한 동사감, 남성적 |

### 지혜·배움(Wisdom/Learning)
| 이름 | 의미 | 메모 |
|---|---|---|
| **Sapio / Sapiens** | 라틴 '알다·현명한 인간' | 사고·문명 동시, 신뢰감. Sapiens는 서적명 연상 |
| **Sophos / Sophia** | 그리스 '지혜' | 우아하나 교육계 흔함(차별↓) |
| **Athena** | 지혜·문명(아테네)의 여신 | 임팩트 크나 매우 흔함(상표 혼잡) |
| **Mentis** | 라틴 '정신·사고의' | 지적, 다소 딱딱 |

### 읽기·문해(Reading/Literacy)
| 이름 | 의미 | 메모 |
|---|---|---|
| **Litera / Literra** | 라틴 '글자·문해' | 직관적, 평범할 수 있음 |
| **Lexio** | 그리스 lexis '말·어휘' | ⚠️ Lexia(문해력 기업)와 유사—충돌 주의 |
| **Logos** | 그리스 '말·이성·논리' | 논리 문해력에 정합, 일반적 |
| **Verba** | 라틴 '말들' | 짧고 부드러움 |

### 발견·성장(Discovery/Growth) — 소크라틱 결
| 이름 | 의미 | 메모 |
|---|---|---|
| **Eureka** | '알아냈다'(스스로 발견) | 소크라틱 정신 직결. 흔하고 수학게임 연상 |
| **Aurora** | 여명·문명의 새벽·빛 | 아름다우나 흔함 |
| **Ascenta** | ascend '올라서다' | 성장·시대 상승감 |

### 코인드 블렌드(합성·독창)
| 이름 | 조합 | 메모 |
|---|---|---|
| **Civora** | Civ(문명)+-ora | 문명 직결. ⚠️ 'Civ'=Sid Meier's Civilization 연상/상표 주의 |
| **Sapira** | Sapiens+-a | 부드러운 브랜드감, 여성적 |
| **Litos** | Litera+Logos | 문해+논리 축약, 유니크 |
| **Aevora** | Aevum(시대)+aurora(빛) | 시대를 밝힌다, 브랜디드 |
| **Wondra** | wonder(경이) | 발견의 경이, 친근 |
| **Questa** | quest(탐구) | 탐험감. ⚠️ Questia(폐업 서비스) 연상 |

## 3. 네이밍 원칙(적용 기준)
- 세계관 A(시대·진화)와 의미 연결 / 11~15세도 성인도 유치하지 않게 / KR·TW(번체)·VN 발음 용이 / 2~3음절 / 도메인·상표 확보 가능성.

## 4. 다음 단계
1. 유저가 3~5개 선호안 압축 → 2. 상표·도메인(.com/.ai) 예비 검토 → 3. 로고·워드마크 시안 → 4. 페르소나·세계관 카피와 정합.

> 결정 요청: 마음에 드는 후보 3~5개를 골라주시면 상표 관점 리스크와 함께 압축·발전시키겠습니다.
