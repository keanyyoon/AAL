# QA 체크리스트 (QA Checklist) — 파일럿 개발 검수

- 작성일: 2026-07-01 · 작성: AAL 검증팀 · 용도: 개발 완료 시 통과 기준

## 1. 교육 타당성 (소크라틱)
- [ ] AI가 어떤 경로로도 정답/완성 문장을 직접 제공하지 않는다(레드팀 시나리오 통과).
- [ ] 오답 시 정답 대신 근거 문장 회귀 유도.
- [ ] 정체 감지 → 비계 사다리 단계 상향 → 그래도 막히면 격려·난이도 조정(방치 없음).
- [ ] Endorse는 능력이 아닌 노력·과정·전략을 칭찬.

## 2. 기능 (P0 4화면 + 추가)
- [ ] 로그인 시각 기록. [ ] 온보딩 페르소나·관심사 저장. 
- [ ] 세션: 지문·문항·대화 통합, 진행 표시, 반응형(PC/태블릿/모바일).
- [ ] 완료 버튼 → 세션 완료 + 전체 로그 트랜잭션 확정.
- [ ] SE 리포트 3블록(배운 것/좋아진 점/보강점)이 실제 로그 근거로 생성.
- [ ] 성장맵/리워드(Lumen·스트릭·3축) 표시.
- [ ] 학부모 데일리 로그 자동 갱신, 4주 추이 차트 + AI 코멘트 + 외부준거 매핑.
- [ ] 성과 알림만 발송(데일리 무알림).

## 3. 데이터 무결성 (4대 데이터)
- [ ] 세션 완료율 산출 가능(완료 이벤트 정확). [ ] 모든 대화·세션에 persona_id 태깅.
- [ ] 리포트 열람·섹션 체류·알림 오픈 로깅. [ ] 문항 유형·난이도·정오답·시간 로깅.
- [ ] 데이터 유실/중복 없음(재접속·에러 시 무결).

## 4. 안전 · 개인정보 (아동 대상)
- [ ] 아동 개인정보 최소 수집·가명화·보호자 동의·열람권.
- [ ] 부적절 주제 차단, 자기비하·경쟁 유발·차별 표현 필터.
- [ ] 아이스브레이킹 주제 연령 적합.

## 5. 접근성 (WCAG 2.1 AA)
- [ ] 색대비 4.5:1↑, 색 외 형태·라벨 병행(3축 컬러). [ ] 키보드 내비·포커스.
- [ ] 터치 타깃 ≥44px. [ ] 듣기 활동 자막. [ ] 애니메이션 감소 옵션.

## 6. 반응형/성능
- [ ] 태블릿·PC·모바일 레이아웃 검증. [ ] 대화 응답 스트리밍 지연 허용치 내.

## 7. 확장 훅 (저비용 선반영 확인)
- [ ] i18n 분리 [ ] 사고요소 언어독립 층 [ ] 로그 subject/language 필드 [ ] 외부준거 매핑 테이블 [ ] 성장로그 학생단위 영속 [ ] 페르소나-세계관 추상화.
